Docker基本命令

Docker命令分为两大类：客户端命令和服务端命令。前者主要是操作接口，后者用来启动Docker daemon。

客户端命令基本格式：docker [OPTIONS] COMMAND [arg...]

服务端命令基本格式：docker daemon [OPTIONS]
